# Challenge 01: The Oregon Trail
Make a simple game, the oregon trail using python and using text only. no gfx.


## File Descriptions

- `screens.py`: Handles the display of different screens and menus in the game.
- `data.py`: Manages game data, including resources, events, and game states.
- `kb.py`: Contains functions for keyboard input handling.
- `oregon_trail.py`: The main game file that initializes and runs the game loop.
- `player.py`: Defines the player character and their attributes.
